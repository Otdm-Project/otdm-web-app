## スタート

```shell
otdm up
```

## トンネルを切断

```shell
otdm down
```

## 接続情報の表示

```shell
otdm status
```

## バージョンの確認

```shell
otdm --version
```

## ヘルプ

```shell
otdm
```

```shell
otdm --help
```
